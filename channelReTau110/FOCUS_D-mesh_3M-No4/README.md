## ベンチマーク情報

* 格子数(格子数倍率)：2995200(8)
* MPI数(ノード数)：20(1),40(2),80(4),160(8),240(12),320(16),400(20),480(24)
* その他特記事項:
  * Dシステムの短時間キュー(d006h,d012h)を用いると，Eシステムが割当て
    られる場合があり，Eシステム上の実行では極端に遅くなる場合があったの
    で，Dシステムのみに割りあてるよう長時間キュー(d096h)を用いて測定した．
